## Building
Add OpenAI config properties in `local.properties`
```local.properties
openai.api.key=
openai.org.id=
openai.project.id=
```
## SwiftExport
To use SwiftExport instead of ObjCExport add to `local.properties`
```bash
kotlin.experimental.swift-export.enabled=true
```
